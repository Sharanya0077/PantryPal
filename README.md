# PantryPal

Smart Recipe & Pantry Manager built using React, Node.js, Express, and MongoDB.

## Features
- Add recipes
- Search recipes by ingredients
- Save favorite recipes
- Generate shopping lists
- Edit and delete recipes

## Tech Stack
### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
